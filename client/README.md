# LINKER Client

A React application built with Vite, TypeScript, shadcn-ui, and Tailwind CSS.

## Project Setup

Install dependencies:
```sh
npm install
```

## Development

Start the development server:
```sh
npm run dev
```

## Build

```sh
npm run build
```

## Technologies

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn-ui
